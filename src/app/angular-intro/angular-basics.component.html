4.5. Create a component

Components are a concept used in most Javascript Frameworks, including Angular. Create your own Angular component and insert it below, by:
  1. Open a new terminal instance by clicking on the + icon
  2. Use the cd src/app command to move into the app directory. You can check that this is successful by using the pwd command. Then use ng generate component random-stuff to create a component called RandomStuff.
  3. Angular will generate the necessary files for a basic component. Go to random-stuff/random-stuff.component.ts to have a look. The selector app-random-stuff is the HTML tag for your new component.
  4. Put <app-random-stuff> below to insert your component into the page!
  5. Note: You modify the HTML of the component in random-stuff.component.html, the same way as in previous exercises.

4.6. Use a NgBootstrap Component - Typeahead

Component libraries speed up the front-end development process by providing pre-built components that can be used with (relatively) little effort. We will use the NgBootstrap Typeahead component here, which allows users to select a predefined input based on what they've already entered.
  1. The NgBootstrap library separates each of its available components into a separate NgModule, so we need to import the NgbTypeaheadModule module from @ng-bootstrap/ng-bootstrap in app.module.ts (the top-level module of our application). This way, we can access the component in random-stuff.component.html
  2. We will now copy the 'Simple Typeahead' example from the NgBootstrap documentation (once we have this in, we can modify as we wish). Go to the Typeahead documentation, view the code for the Simple Typeahead, and copy the states const into random-stuff.component.ts, directly underneath the imports.
  3. Next copy the search: OperatorFunction... property from the documented example, into RandomStuffComponent in random-stuff.component.ts. You may need to add the import statements import { Observable, OperatorFunction } from 'rxjs' and import { debounceTime, distinctUntilChanged, map } from 'rxjs/operators'. This object defines how the typeahead will behave - after 200ms from the user's last keystroke, if there are more than two characters, the object searches through the states object for strings which contain the characters, and returns the first 10.
  4. Time to add the text input, and attach our Typeahead feature. Add an input element to random-stuff.component.html, and also add [ngbTypeahead]="search" to the tag. This invokes the NgbTypeaheadModule, and passes in the search object to define how the Typeahead should behave.
  5. Try out your new typeahead!
